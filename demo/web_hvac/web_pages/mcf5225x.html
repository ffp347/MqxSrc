MCF5225X Family

This 32-bit device is based on the Version 2 (V2) ColdFire reduced instruction set computing (RISC) core with an enhanced multiply-accumulate unit (EMAC) and hardware divider providing 76 Dhrystone 2.1 MIPS at a frequency up to 80 MHz. On-chip modules include the following:
  • V2 ColdFire core with enhanced multiply-accumulate unit (EMAC)
  • Up to 64 Kbytes of internal SRAM
  • Up to 512 Kbytes of on-chip flash memory
  • Three universal asynchronous receiver/transmitters (UARTs)
  • Two inter-integrated circuit (I2C) bus controller
  • 12-bit analog-to-digital converter (ADC)
  • Queued serial peripheral interface (QSPI) module
  • Four-channel, 32-bit direct memory access (DMA) controller
  • Four 32-bit general purpose timers with optional DMA support
  • Two 16-bit periodic interrupt timers (PITs)
  • Programmable software watchdog timer
  • Two interrupt controllers capable of handling up to 128 interrupt sources
  • Clock module with 8 MHz on-chip relaxation oscillator and integrated phase-locked loop (PLL)
  • External Mini-FlexBus interface
  • Fast Ethernet Controller
  • Cryptographic Acceleration Unit
Attach the included USB Flash drive to your board to browse the web pages on there and get more information on the MCF5225X Family.
